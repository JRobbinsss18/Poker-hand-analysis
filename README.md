# Poker Hand Analysis

This repository contains tools for simulating, evaluating, and analyzing Texas Hold'em poker hands. It enables users to set up poker hands, modify player and community cards, and simulate win probabilities against various numbers of opponents.

## Table of Contents

- [Project Overview](#project-overview)
- [Files in the Repository](#files-in-the-repository)
- [Installation and Usage](#installation-and-usage)

## Project Overview

The objective of this project is to provide a comprehensive tool for Texas Hold'em players to analyze poker hands. Users can:

- Set up poker hands by specifying player and community cards.
- Simulate win probabilities against different numbers of opponents.
- Visualize statistical data from simulated games.

## Files in the Repository

- `poker_analysis.py`: Main script for setting up and analyzing poker hands.
- `CSV_to_visuals.ipynb`: Jupyter notebook for generating visualizations from CSV data.
- `stats_2_players.csv` to `stats_7_players.csv`: CSV files containing statistical data for games with 2 to 7 players.
- `README.md`: This file, providing an overview and instructions for the project.
